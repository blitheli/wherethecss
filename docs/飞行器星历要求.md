# 开发飞行器星历的类或组件

当前项目中,飞行器的位置在世界坐标系中是固定的（经纬度），实际上空间飞行器是运动的，由随时间变化的位置和速度确定（简称星历）

帮我查询并确定以下技术路线，然后进行相应功能的开发。

- 在THREE, R3F中，是否有类似Cesium中Entity类的对应功能，可直接赋值Position属性那种的？
- 若没有的化，借助Cesium库和satellite.js库实现卫星的星历类
- 飞行器的星历先采用两行根数，内部函数: Update(time),返回ECEF(ECI)中的位置和速度
- 在R3F UseFrame中，每帧调用星历对应的Update(time)函数
- 更新Tiangong.tsx，打开页面时，每一帧都默认当前系统时间在运行
- 两行根数先默认给一个（可从网上查询获取中国空间站的TLE）

开发的过程，要点，技术路径都写清楚，存放在md文档中
