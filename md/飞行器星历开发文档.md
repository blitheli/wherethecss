# 飞行器星历功能开发文档

## 开发日期
2026-03-31

## 开发目标
实现基于两行根数（TLE）的卫星星历计算系统，支持实时和仿真两种时间模式，替代原有的固定经纬度位置渲染。

---

## 技术路线

### 1. TLE计算库选择
- **库**: `satellite.js`
- **功能**: 解析两行根数，计算卫星在ECI（地心惯性惯性）坐标系中的位置和速度
- **坐标系**: 返回ECI坐标系（J2000历元），需转换为ECEF（地心固连）坐标系

### 2. THREE/R3F集成方式
- **现状**: R3F没有类似Cesium Entity的自动位置更新机制
- **方案**: 每帧调用星历Update函数，手动更新模型位置
- **实现**: 通过forwardRef获取模型Group对象，直接设置position属性

### 3. 坐标系转换链
```
TLE → satellite.js → ECI(km) → ECEF(km) → World(m)
       ↑
       |  旋转矩阵（格林尼治恒星时）
       |
    gmst(Date)
```

## 实现细节

### 1. 卫星星历类 (`app/core/SatelliteEphemeris.ts`)

#### 核心方法
- **constructor(tleLine1, tleLine2)**: 解析TLE初始化卫星对象
- **update(date)**: 计算给定时间的ECI位置和速度（返回km单位）
- **getPositionECEF(date, matrixWorldToECEF)**: 返回ECEF位置（m单位）

#### 坐标系转换逻辑
```typescript
// 1. ECI到ECEF旋转矩阵（格林尼治恒星时）
gmst = gstime(date)
rotationMatrix = [
  cos(gmst), sin(gmst), 0,
  -sin(gmst), cos(gmst), 0,
  0, 0, 1
]

// 2. 应用变换
positionECEF = positionECI × rotationMatrix × matrixWorldToECEF × 1000
```

### 2. 时间控制组件 (`app/components/TimeControl.tsx`)

#### 功能特性
- **实时/仿真切换**: Switch开关控制模式
- **时间进度条**: Slider显示和设置当前时间（0-1000代表一天）
- **加速/减速**: 按钮/预设倍速（1x, 10x, 60x, 300x, 900x, 3600x）
- **暂停/继续**: 实时模式下不可用

#### 实现方式
```typescript
// 实时模式：requestAnimationFrame同步系统时间
useEffect(() => {
  if (isRealtime) {
    const update = () => {
      onTimeChange(new Date())
      requestAnimationFrame(update)
    }
    update()
  }
}, [isRealtime])

// 仿真模式：根据simulationSpeed累加时间
useEffect(() => {
  if (!isRealtime && !isPaused) {
    const update = () => {
      elapsed = (now - lastUpdate) / 1000
      simulatedTime += elapsed × simulationSpeed
      onTimeChange(simulatedTime)
      requestAnimationFrame(update)
    }
    update()
  }
}, [isRealtime, isPaused, simulationSpeed])
```

## 两行根数(TLE)说明

### 格式结构
```
行1: 1 NNNNNU NNNNNAAA NNNNN.NNNNNNNN +.NNNNNNNN +NNNNN-N +NNNNN-N N NNNNN
行2: 2 NNNNN NNN.NNNN NNN.NNNN NNNNNNN NNN.NNNN NNN.NNNN NN.NNNNNNNNNNNNNN
```

### 字段含义
- **卫星编号**: NORAD目录号（天宫：58020）
- **历元**: TLE有效时间参考点
- **平均运动**: 轨道周期（rev/day）
- **倾角**: 轨道面与赤道面夹角（天宫约41.5°）

### 中国空间站TLE示例
```javascript
{
  line1: '1 58020U 24091A   26090.50000000  .00005200  00000-0  22992-3 0  9994',
  line2: '2 58020  41.4725 102.0234 0004619 296.5318 189.3824 15.49053630 12345'
}
```

### TLE数据源
- **Celestrak.org**: https://celestrak.org/NORAD/elements/space-stations.txt
- **更新频率**: 建议每周更新一次以保持精度

## 技术要点记录

### 坐标系约定
- **ECI**: 地心惯性坐标系（J2000历元），卫星计算结果
- **ECEF**: 地心固连坐标系，随地球自转，渲染使用
- **World**: Three.js世界坐标系，单位为米

### 时间处理
- **Date对象**: JavaScript时间戳（毫秒）
- **gmst**: 格林尼治恒星时，用于ECI→ECEF转换
- **单位转换**: satellite.js输出km需×1000转换为米

### 性能优化
- **星历对象缓存**: useResource避免重复创建
- **向量复用**: 使用class成员变量避免GC
- **帧同步**: requestAnimationFrame与R3F渲染循环解耦
